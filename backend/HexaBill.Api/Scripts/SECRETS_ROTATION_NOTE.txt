HexaBill – rotate credentials if exposed
========================================

If DATABASE_URL, DB_PASSWORD, RENDER_API_KEY, VERCEL_TOKEN, or JWT secrets were shared (chat, ticket, screenshot), rotate them:

1. Render Dashboard: Postgres → reset password; update local .env and Render web service env.
2. Render: Account Settings → API Keys → revoke and create new key if needed.
3. Vercel: Project or account token → revoke and create new.
4. JWT: set JwtSettings__SecretKey to a new value (e.g. openssl rand -base64 48) and redeploy so sessions re-login.

Never commit .env; keep it gitignored.
